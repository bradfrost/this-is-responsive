The Fluid Accordion

We've seen them around the block once or twice. It's the ol' accordion panel widget, but done in a responsive manner

Pros

  • » You don't have to keep building it over and over again.
  • » The width is fluid using percentages
  • » Semantic class names
  • » Builds on "Best Practices" for hidden elements
  • » Content Panels are still accessible without Javascript
  • » User feedback for "active" panel states
  • » Clean foundation making it easy to build upon
  • » Works on iPhone, Android, Opera Mobile Emulator
  • » Makes use of ARIA attributes. Using Javascript we adjust the accordion panels active/inactive states

Cons

  • » Tied down by jQuery
  • » Minimal Keyboard events (we'll let you do that yourself)
  • » Not built for touch support
  • » Tied down "tabindex" attribute values; you gotta change em yo' self
